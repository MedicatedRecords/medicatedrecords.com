# Medicated Records

Website for the Medicated Records music label — artist profiles, DJ mixes, releases,
events, and press kits. Built with [Astro](https://astro.build) and Tailwind CSS, hosted on
Vercel.

## Planning & conventions

- **[AGENTS.md](AGENTS.md)** — source of truth: stack, content model, design system, and
  working rules for agents and contributors. (**[CLAUDE.md](CLAUDE.md)** points here.)
- **[docs/FEATURES.md](docs/FEATURES.md)** — the initial feature-set spec.
- **[sprints/](sprints/)** — the planning system and running feature list. Start with
  [sprint-01-foundation.md](sprints/sprint-01-foundation.md).

## Getting started

> The Astro project is scaffolded in Sprint 01. Once it exists:

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run preview
```
